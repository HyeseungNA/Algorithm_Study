## 초기 세팅(중요)
1. 스터디 Repository로 들어가 우측 상단의 `Fork` 버튼 클릭

2. 본인의 Repository에 스터디 Repo가 생성

3. 본인의 스터디 Repository로 들어가 주소 복사 및 로컬 폴더에 clone 진행 

4. 로컬 Repository에 본인 이름의 폴더 생성

5. 해당 본인 폴더내에 스터디 인증 파일을 넣고 `commit` - `push` 하면 인증 성공</br>
